About the Project

AI/ML Engineer Assignment: Building a RAG pipeline for mathematical question extraction from RD Sharma Class 12

Project Overview

This project implements a Retrieval-Augmented Generation (RAG) pipeline to extract mathematical questions from RD Sharma Class 12 textbook using advanced AI techniques.

Problem Statement

Automatically extract and format mathematical questions from RD Sharma Class 12 PDF, handling various text formats, mathematical notation, and ensuring high accuracy.

Solution Approach

RAG pipeline combining pattern matching, OCR, and LLM validation for robust question extraction with confidence scoring from a fixed textbook source.

Technical Architecture

Core Components
  • Fixed PDF Source: RD Sharma Class 12 textbook
  • OCR Support: Tesseract + EasyOCR fallback
  • LLM Integration: GPT-2 via Hugging Face
  • LaTeX Generation: Professional formatting
Key Features
  • Confidence Scoring: Multi-factor quality assessment
  • Deduplication: Remove duplicate questions
  • Pattern Matching: Regex-based extraction
  • Testing: Comprehensive unit tests

How to Use

Web Interface
  1. Select chapter number (1-100)
  2. Enter topic name (e.g., "Conditional Probability")
  3. Click "Extract Questions"
  4. View results and download LaTeX file
Command Line
  1. Navigate to project directory
  2. Run: python rd_sharma_pipeline.py --chapter 30 --topic "Conditional Probability"
  3. Check output_tex_files/ for results
